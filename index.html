Levenshtein Edit Distance: Algorithmic Approaches And Data Structures For Text Processing

Auguest 2023 - December 2023 Project Link

  • At the heart of computational linguistics lay the Levenshtein Edit Distance algorithm, which was crucial for precise text processing. Our project compared the runtime of dynamic versus recursive implementations, emphasizing their use in text applications.
  • Developed by Vladimir Levenshtein, the algorithm measures string similarity through minimal edits, influencing fields like spell-checking and DNA analysis.
  • We also explored data structures—Hash Set, Trie, and Binary Search Tree—for quick word retrieval before calculating the Levenshtein Distance. Recursive methods had a time complexity of O(Max(N, M)^3), while dynamic programming was more efficient at O(N*M).
  • Hash Sets proved to be the fastest for word verification, while Binary Search Trees were slower due to imbalances. Our project highlighted how combining these data structures and algorithms optimized Levenshtein Distance for real-world use.
